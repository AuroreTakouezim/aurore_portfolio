# Portfolio – Aurore Takouezim

Ce dépôt contient mon site de portfolio déployé avec **GitHub Pages**.

## Structure
- `index.html` – page principale
- `styles.css` – styles (thème sombre/clair, responsive)
- `script.js` – interactions (menu mobile, thème)
- `assets/img/favicon.svg`

## Déploiement (Pages)
Paramètres → Pages → *Deploy from a branch* → `main` / `/` (root).
L’URL sera : `https://auroretakouezim.github.io` ou `https://auroretakouezim.github.io/` selon votre configuration.
